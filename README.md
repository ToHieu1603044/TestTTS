Đây là một ứng dụng quản lý công việc đơn giản sử dụngg:

- React 18
- TypeScript
- React Hook Form
- React Router v6
- LocalStorage (lưu dữ liệu tạm)

---

Hướng dẫn cài đặt

### Yêu cầu
- Node.js >= 18
- npm hoặc yarn

### Cài đặt

# Clone dự án
git clone https://github.com/yourusername/task-manager.git

# Cài đặt dependencies
cd my-app
npm install
# hoặc: yarn
